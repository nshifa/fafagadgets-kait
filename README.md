## 1. Identitas Bisnis & Value Proposition
* **Nama Bisnis:** FafaGadgets
* **Tagline:** *"Future Technology, Delivered Today"*
* **Deskripsi Bisnis:**  
  FafaGadgets adalah platform e-commerce ritel khusus produk elektronik dan gadget premium di Indonesia. Kami menyediakan smartphone, laptop, smartwatch (wearables), casing pelindung, serta aksesoris teknologi lainnya. FafaGadgets berupaya memecahkan kekhawatiran terbesar konsumen e-commerce elektronik di Indonesia, yaitu maraknya produk palsu (refurbished/rekondisi) dan risiko kerusakan barang bernilai tinggi selama proses pengiriman kurir logistik.
* **Value Proposition:**
  1. **100% Guaranteed Originality:** Menjamin semua barang baru bersegel resmi dan bergaransi resmi distributor nasional selama 1 tahun.
  2. **Extra-Secure Transit Packaging:** Proteksi pengiriman ekstra menggunakan pembungkus gelembung tebal berlapis dan kotak kardus kokoh secara gratis.
  3. **7-Day Replacement Policy:** Jaminan penggantian unit baru secara instan apabila terjadi cacat produksi dari pabrik dalam 7 hari pertama setelah barang diterima.
  4. **Direct Admin Communication:** Menghubungkan langsung pembeli ke admin via WhatsApp/Instagram DM saat stok barang habis untuk pre-order, guna mencegah hilangnya pembeli (*lead abandonment*).

## 3. Analisis Pasar & Kompetitor
### Analisis SWOT
* **Strengths (Kekuatan):**
  * Jaminan garansi resmi distributor nasional selama 1 tahun.
  * Proteksi pengemasan premium gratis (mengurangi risiko klaim kerusakan barang pecah).
  * Integrasi CS DM Instan untuk mengatasi penundaan pembelian akibat stok kosong.
  * Visualisasi *Order Tracking* (pelacakan lokasi) pasca checkout untuk keterbukaan logistik.
* **Weaknesses (Kelemahan):**
  * Margin keuntungan per unit relatif tipis pada produk utama (laptop & HP) karena persaingan harga pasar yang ketat.
  * Ketergantungan tinggi pada stok dari distributor resmi pemegang merek.
* **Opportunities (Peluang):**
  * Meningkatnya tren kebutuhan laptop ringkas dan HP handal untuk sistem kerja hibrida (*hybrid working*).
  * Kebutuhan pelanggan akan transparansi posisi kurir pengiriman secara digital.
* **Threats (Ancaman):**
  * Persaingan ketat dengan peritel fisik raksasa yang juga merambah e-commerce (e.g. Eraspace/Erafone, ibox).
  * Depresiasi nilai harga barang elektronik yang cepat karena peluncuran tipe baru yang konstan.

### Analisis Kompetitor
1. **Eraspace / Tokopedia Official Store (Direct Competitor):**
   * *Kelebihan:* Memiliki ribuan gerai fisik (omnichannel), modal sangat besar, dan kerja sama langsung sebagai distributor utama merek besar (seperti Apple dan Samsung).
   * *Kelemahan:* Antarmuka web terkadang lambat karena beban aset besar; proses klaim garansi online terkadang memakan waktu lama.
   * *Diferensiasi FafaGadgets:* Kami fokus pada **kecepatan akses situs web**, layanan pengemasan aman berasuransi tanpa biaya tambahan, kebijakan **tukar baru 7 hari** jika cacat pabrik tanpa prosedur birokrasi berbelit, dan **opsi pre-order langsung ke admin** via WA/Instagram saat stok web habis.

## 5. Model Bisnis & Revenue Stream
* **Model Bisnis:** B2C E-Commerce Retail
  * FafaGadgets menjual produk elektronik secara langsung ke konsumen akhir melalui platform e-commerce resmi. Kami menjalin kemitraan dengan importir resmi untuk mendapatkan harga grosir kompetitif.
* **Revenue Streams (Aliran Pendapatan):**
  1. Penjualan utama perangkat keras bernilai tinggi (*high-ticket items*) seperti laptop dan smartphone.
  2. Penjualan aksesoris gadget (casing, keyboard, charger, powerbank). Meskipun volume harga aksesoris lebih rendah, **margin keuntungan aksesoris jauh lebih tinggi (bisa mencapai 40-50%)** dibandingkan laptop/HP yang hanya berkisar 5-8%. Penjualan silang (*cross-selling*) aksesoris ini menjadi penopang keuntungan bisnis utama.

## 6. Rencana Manajemen Stok & Strategi Pre-Order Kontak Admin
Dalam dunia ritel gadget, kehabisan stok (*out-of-stock*) adalah salah satu penyebab utama hilangnya pembeli ke kompetitor. Untuk itu, FafaGadgets mengadopsi taktik **Direct Admin Message Conversion**:
- **Indikator Stok Dinamis:** Menampilkan label warna yang menstimulasi pembelian (Hijau: Tersedia, Oranye: Kritis / Sisa 1 Unit, Merah: Habis).
- **Tombol Hubungi Admin:** Bila stok barang 0 (Habis), tombol beli otomatis diganti dengan tombol chat bertuliskan **"Hubungi Admin via WA/IG"**.
- **Pesan Otomatis (Pre-filled):** Mengarahkan user membuka chat WhatsApp dengan template pesan: `"Halo Admin FafaGadgets, saya ingin menanyakan restock untuk produk [Nama Barang] yang sedang habis."`
- **Rasionalitas Bisnis:** Menjaga pembeli tetap berada di dalam corong penjualan (*sales funnel*). Admin dapat menyarankan pre-order dengan uang muka (DP), menawarkan alternatif unit lain yang setara, atau mencatat kontak pelanggan untuk dihubungi segera saat restock tiba.

## 7. Sistem Pelacakan Lokasi Pengiriman (Order Tracking Navigation)
Kepercayaan pelanggan elektronik sangat dipengaruhi oleh transparansi proses logistik setelah pembayaran lunas. FafaGadgets mengintegrasikan fitur **Pelacakan Posisi Paket Interaktif**:
- **Status Alur Pengiriman:**
  1. **Pesanan Diterima:** Pembayaran sukses, pesanan masuk antrean sistem.
  2. **Diproses:** Paket diperiksa serial number-nya (IMEI/S/N) dan dibungkus proteksi kayu/bubble wrap di gudang Jakarta.
  3. **Transit Hub:** Paket dalam perjalanan logistik dan disortir di Hub Transit FafaExpress.
  4. **Kurir Mengantar:** Kurir (nama/HP tertera) sedang membawa paket menuju alamat tujuan.
  5. **Sampai Tujuan:** Paket sukses diserahterimakan kepada pembeli.
- **Simulator Perjalanan:** Menyediakan tombol simulasi cepat (*speed-up*) untuk mempercepat linimasa status guna keperluan pengujian/penilaian dosen.

## 9. Rencana Data Analytics untuk Pengambilan Keputusan
FafaGadgets melacak siklus perjalanan pelanggan melalui integrasi visual debugger GA4:

* **Event yang Dilacak:**
  * `page_view` (Halaman dimuat).
  * `view_item` (Modal spesifikasi gadget dibuka).
  * `add_to_cart` (Gadget masuk keranjang belanja).
  * `begin_checkout` (User masuk ke form pengiriman).
  * `purchase` (Transaksi pembayaran disimulasikan sukses).
* **Metrik Utama yang Dipantau:**
  1. **Average Order Value (AOV):** Mengukur rata-rata nominal belanja. Keberhasilan cross-selling casing dan charger dapat terlihat dari kenaikan AOV.
  2. **Cart Abandonment Rate:** Persentase keranjang belanja yang ditinggalkan. Jika tingkat pengabaian tinggi, evaluasi kemudahan pengisian form checkout perlu dilakukan.

## 10. Panduan Teknis & Cara Menjalankan Website
### Menjalankan di Komputer Lokal (Localhost):
1. Unduh atau clone repositori proyek ini.
2. Buka berkas `index.html` dengan cara melakukan klik ganda (double-click) di browser, ATAU gunakan ekstensi **Live Server** di VS Code untuk memuatnya sebagai localhost.

### Cara Deploy ke GitHub Pages:
1. Buat repositori baru di akun GitHub Anda (misal: `fafagadgets`).
2. Hubungkan repositori lokal Anda ke GitHub dengan perintah:
   ```bash
   git remote add origin <URL_REPOSITORI_GITHUB_ANDA>
   git branch -M main
   git push -u origin main
   ```
3. Di halaman repositori GitHub Anda, masuk ke menu **Settings** > **Pages**.
4. Di bagian **Build and deployment**, pilih branch **main** (folder `/root`), lalu klik **Save**.
5. Tunggu 1-2 menit hingga GitHub selesai memproses. Website Anda akan online di URL: `https://<username>.github.io/fafagadgets/`.
